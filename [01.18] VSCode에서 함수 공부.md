## 1. syslib

- 프로그램을 실행할 때 옵션 주기

```python
import sys

try : 
    if sys.argv[1] == 'call' :
        print(sys.argv[1])
    elif sys.argv[1] == 'exit' :
        sys.exit()
    elif sys.argv[1] == 'path' :
        print(sys.path)
    elif sys.argv[1] == 'append_path' :
        sys.path.append(sys.argv[2])
        print(sys.path)
    else :
        print('Not support')
except Exception as e :
    print('write more data!')

print('finish!!')
```

## 2. pickle _ `dump( )` & `load( )` 

1. 객체를 파일로 저장

```python
import pickle

with open('pickle_test.txt', 'wb') as f : ## wb : write & use data
    data = {1 : 'python', 2 : 'java'}
    pickle.dump(data, f)
```

2. 객체를 불러오기

```python
# import pickle

with open('pickle_test.txt', 'rb') as f :
    data = pickle.load(f)
    print(data)
  
# {1: 'python', 2: 'java'}
```

## 3. OS 함수

```python
import os 

print(os.environ)

# 경로 이동
print(os.getcwd())
os.chdir(os.getcwd()+'/lib')
print(os.getcwd())

# 목록 보기
print(os.system('ls'))

# 실행하기
f = os.popen('ls')
print(f.read())

# 파일 생성 & 삭제
os.mkdir('lib2')
os.rmdir('lib2')

# 파일 이름 변경
os.mkdir('lib2')
os.rename('lib2', 'lib3')
```
